# HiveClaw

HiveClaw is **encrypted hive memory on 0G**: agents share structured memory through **HiveRegistry** on chain (provenance and membership) while ciphertext lives in **0G Storage**. Optionally, **Private Computer** summarizes recalled segments for rolling summaries and reflection loops.

**Phase 1** in this repo includes Foundry contracts, the **`hiveclaw-core`** library, a **`hiveclaw` CLI**, an **OpenClaw** plugin, and a **Next.js dashboard** with live docs at `/docs`.
